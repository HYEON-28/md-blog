# Feature: 메인 대시보드

**Status:** 구현 완료

## 목적
로그인한 사용자가 연동 레포, 블로그, 오늘의 코드 변경 현황을 한눈에 파악한다.

## 화면 구성 (`/main`)

### 상단 인사 영역
- 오늘 날짜 (한국어 요일 포함)
- "안녕하세요, {githubUsername} 님"

### 통계 카드 (3개)
| 카드 | 데이터 소스 | 색상 |
|------|------------|------|
| 연동된 레포지토리 수 | GET /api/repos/connected | 초록 |
| 블로그 연동 레포 수 | 위 응답에서 blog=true 필터 | 보라 |
| 오늘 수정된 파일 수 (+/- 라인) | GET /api/repos/today-updates | 노랑 |

### 섹션 1: 레포지토리 관리
- 연동된 레포 목록 (언어 색상 dot, 이름, 설명, 언어 태그, 상대 시간)
- 헤더 우측 "레포 설정" 버튼 → `/repoSettings`
- 접기/펼치기 토글

### 섹션 2: 블로그 관리
- `blog=true` 인 레포만 표시
- 헤더 우측 "블로그 설정" 버튼 → `/blogSettings`
- 접기/펼치기 토글

### 섹션 3: 오늘의 업데이트
- 레포별 그룹 (레포명, 언어 색상, +/- 라인 합계, 파일 수)
- 레포 클릭 → 파일 목록 펼침
- 파일 클릭 → `/file-updated` 로 이동 (state로 repoFullName, filePath 전달)
- 파일 상태 dot: added(초록), modified(파랑), deleted(빨강)
- 접기/펼치기 토글
